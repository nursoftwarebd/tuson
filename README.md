## Installation

Install tuson with npm

```bash
  git clone
  cd tuson
  npm install  
```
    
## Styling with tailwind css

- Run this command for updating any styles.
```bash
npx tailwindcss -i ./css/input.css -o ./dist/output.css --watch
```
